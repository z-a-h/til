# TMUX COMMANDS

### Create new panes with vertical splits
>`C-b %`

### Create new pane with horizontal split
>`C-b "`

### Navigate between panes
>`C-b` + arrow direction

### Close panes
>`exit` or `Ctrl-d`

### View available panes
>`C-b ?`

### Create new window in a session
>`C-b c`

### Navigate to next window
>`C-b n`

### Navigate to previous window
>`C-b p`

### Detach from session
>`C-b d`

### Detach from specific session
>`C-b D`

### View all running sessions
>`tmux ls`

### Reattach to a session
>`tmux attach -t`

### Create a new named session
>`tmux new -s`

### Rename a session
>`tmux rename-session -t` OR `C-b $`

### Scroll through list of session
>`C-b s`

### Zoom in on a pane
>`C-b z` 

### Resize pane
>`C-b :resize-pane -[L, R, U, or D]'

### Enter Copy Mode
>`C-b [`

### Set Copy Mark
>`Space`

### Execute Copy
>`Enter`

### Pase Selected Text
>`C-b ]`

### Enable Mouses
>`set mouse on`
